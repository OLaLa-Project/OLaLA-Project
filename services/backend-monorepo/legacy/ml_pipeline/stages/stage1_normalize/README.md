# Stage 1: Normalize (텍스트 정규화)

> **담당**: Team A (이윤호, 성세빈)

## 역할
사용자 입력 텍스트를 정규화합니다.

## 입력/출력

| 구분 | 필드 | 설명 |
|------|------|------|
| 입력 | `raw_claim` | 원본 텍스트 |
| 출력 | `normalized_claim` | 정규화된 텍스트 |
| 출력 | `language` | 감지된 언어 |

## 구현 체크리스트

- [ ] 특수문자 제거/정규화
- [ ] 공백 정리
- [ ] 언어 감지
- [ ] 입력 길이 검증
