# Stage 9: Judge (최종 판정)

> **담당**: Common (이은지, 성세빈)

## 역할
통합된 검증 결과를 바탕으로 최종 판정을 내립니다.

## 입력/출력

| 구분 | 필드 | 설명 |
|------|------|------|
| 입력 | `aggregated_result` | 통합 결과 |
| 입력 | `top_evidences` | 증거 리스트 |
| 출력 | `judgment` | 최종 판정 |

## 판정 라벨

| 라벨 | 설명 |
|------|------|
| `TRUE` | 사실로 확인됨 |
| `FALSE` | 거짓으로 확인됨 |
| `MIXED` | 부분적으로 사실/거짓 |
| `UNVERIFIED` | 검증 불가 |
| `REFUSED` | 정책상 처리 거부 |
